Nopayloaddb Documentation
==========================

The nopayloaddb project works as an implementation of the Conditions Database reference for the HEP Software Foundation.
It provides a RESTful API for managing payloads, global tags, payload types, and associated data.
The API allows clients to perform CRUD operations on these resources.

This documentation is under development.
